# Informatyka śledcza

## Email proxy, skanujące pod kątem złośliwych wiadomości

## How to run

To start proxy

```
python run_proxy.py
```

To start receiver

```
python run_receiver.py
```

To start quarantine

```
python run_quarantine.py
```

To send emails

```
python run_sender.py
```

## Settings

All settings can be set in the `email_proxy.settings` file.
